4. The os module

  • The os module (miscellaneous operating system interfaces) provides a portable way of using operating system dependent functionality
  • os.mkdir makes a new directory
  • os.rename moves a file from one directory to another
  • os.listdir lists files in a directory

6. Relative paths, and the current working directory

  • os.getcwd returns the path of the current working directory, this is the shell's equivalent of pwd
  • os.chdir is used to change directories, this is the shell's equivalent of cd

7. os functions and shell commands

  • os.getcwd -> pwd -> what's the current directory?
  • os.listdir -> ls -> what files are here?
  • os.mkdir -> mkdir -> make a new directory
  • os.rename -> mv -> move or rename file or directory
  • Portable code is code that will work correctly on different systems
