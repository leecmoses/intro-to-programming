5. What can go wrong?

  • Non-deterministic means something different could happen each time
  • Network checklist: wi-fi, internet service, server address, server running, server understands request, and code understands response
  • Status code 404 is when the code sent a request to the server, but the server could not find the page
  • Status code 200 is when the code sent a request to the server, and the server successfully sent back the requested data

6. Try and exceptions

  • NameError and TypeError are examples of exceptions, exceptions signals that an error has occurred
  • Error is a generic term while an exception is an error that is detected by Python when running the code
  • Python dects syntax erros before the code runs, so the try and exception statements will not catch the error
  • Python may raise an exception while the code is running
  • When Python raises an exception, the default behavior is for the program to exit (i.e., crash) and display a traceback

7. What is JSON?

  • JSON (JavaScript Object Notation) is just a format - it's a way to structure data so that it's easy to make use of data from code written in many different languages (including Python)
  • JSON is easy for humans to read and write, and it is easy for machines to parse and generate

# Objects and Classes

1. Welcome to objects and classes

  • Object-oriented programming (OOP) is a programming style that is associated with the concepts of Class, Objects and various concepts revolving around these two, like inheritance

2. Review objects and data types

  • One way to tell whether something is an object is to ask whether it can be used as the value of a variable
  • A string, open file, and even a function are objects; but a while loop is not
  • Every objet occupies a space in computer memory; when the object is created, Python makes space for it
  • When variables referring to an object go out of scope, Python can reclaim that memory space by deleting unused objects
  • Not every object can be modified; strings and integers can not be modified, for instance
  • Strings and lists have some method names in common, such as count and find; but the string type and the list type each also have methods the other does not

3. Objects belong to classes

  • isinstance function checks if a object belongs to a class
  • type function returns the class the object belongs to
  • All booleans are in the int and bool classes

4. Python classes versus CSS classes

  • Different languages use same words to refer to slightly different concepts
